# Decisions

## Purpose

This folder stores durable decision records and decision-related references that complement the formal ADR set.

## What Belongs Here

- Decision summaries
- Decision indexes
- Supporting decision references
- Long-lived product or workflow decisions

## Naming Conventions

- Use decision names or identifiers that clearly describe the outcome.
- Keep names stable enough to reference over time.

## Rules

- Do not place raw brainstorming here.
- Keep decisions separate from source material and implementation notes.
- Preserve historical context when documenting a resolved choice.
