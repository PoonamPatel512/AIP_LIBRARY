# Aviation Knowledge

## Purpose

This folder stores aviation-domain knowledge that supports correct interpretation of the product’s metadata model.

## What Belongs Here

- AIRAC concepts
- Aviation publication terminology
- Domain distinctions and definitions
- Aviation standards context

## Naming Conventions

- Use aviation subject names or concept names.
- Keep terminology consistent with the domain.

## Rules

- Do not store generic product notes here.
- Separate aviation meaning from implementation detail.
- Preserve authoritative terminology where possible.
