# Knowledge

## Purpose

This folder stores AI-readable knowledge that has been derived from source material and organized for future reuse.

## What Belongs Here

- Structured summaries
- Domain knowledge notes
- Architecture interpretations
- Implementation guidance
- Reusable reference knowledge

## Naming Conventions

- Use concise names that describe the knowledge domain.
- Prefer stable, topic-based file names over ad hoc session labels.

## Rules

- Do not store original source documents here.
- Keep knowledge separate from raw reference material.
- Ensure content is concise, durable, and easy for AI agents to retrieve.
