# Glossary

## Purpose

This folder stores canonical terminology used by the repository.

## What Belongs Here

- Domain terms
- Product vocabulary
- Acronyms and abbreviations
- Definitions that need stable reuse across documents

## Naming Conventions

- Use alphabetic or concept-based file names when practical.
- Keep entries easy to scan and cross-reference.

## Rules

- Definitions should stay concise and consistent.
- Do not use the glossary for decisions, implementation notes, or source copies.
- Prefer authoritative terminology over invented shorthand.
