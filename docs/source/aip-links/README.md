# AIP Links Source

## Purpose

This folder stores original link inventories, source lists, and other reference material for official AIP and AIS links.

## What Belongs Here

- Official link inventories
- Source link exports
- Link reference spreadsheets or text extracts

## Naming Conventions

- Use names that identify the authority, region, or source set.
- Keep original link list names recognizable when possible.

## Rules

- Do not add interpretation or synthesized summaries here.
- Preserve exact link values and source context.
- Keep the folder limited to link-bearing source material.
