# Source

## Purpose

This folder stores original business and external source documents that ground the repository’s knowledge base.

## What Belongs Here

- Original product source documents
- Official references and source extracts
- Source-linked material that should remain close to its origin

## Naming Conventions

- Use descriptive names that preserve the original document identity.
- Prefer file names that reflect source type and origin.

## Rules

- Do not rewrite source material into summaries here.
- Do not mix source documents with interpretation or architecture notes.
- Keep source material traceable to its origin.
