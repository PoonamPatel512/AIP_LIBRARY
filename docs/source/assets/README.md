# Source Assets

## Purpose

This folder stores original supporting assets that belong to source documents.

## What Belongs Here

- Images
- Diagrams
- Tables
- PDFs or other supporting files tied to source material

## Naming Conventions

- Use asset names that match the parent source document or topic.
- Keep names short, explicit, and stable.

## Rules

- Do not place analysis or summaries here.
- Keep assets tied to their source context.
- Avoid storing unrelated media in this folder.
