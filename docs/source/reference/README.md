# Reference Source

## Purpose

This folder stores reference documents that are useful as source inputs but are not themselves authoritative decisions or summaries.

## What Belongs Here

- External reference material
- Standards excerpts
- Supporting context documents
- Imported reference notes

## Naming Conventions

- Use names that reflect the original reference topic.
- Keep source provenance visible in the file name when practical.

## Rules

- Do not store working drafts or decisions here.
- Keep reference material separate from summaries and analysis.
- Preserve the original meaning and structure where possible.
